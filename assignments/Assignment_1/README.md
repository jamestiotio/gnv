## Setup Instructions

To execute the program, simply run the following commands (where `filename` is the filename of the OBJ file of interest):

```bash
cd Assignment_1_linux/
rm -r build/
mkdir build/
cd build/
cmake ..
make
./Assignment_1 ../data/filename.obj
```
