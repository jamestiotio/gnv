## Setup Instructions

To setup the environment, simply run the following commands:

```bash
cd Assignment_5_linux/
rm -r build/
mkdir build/
cd build/
cmake ..
```

Then, to execute the program, simply run the following commands from the `build` directory:

```bash
make
./Assignment_5
```
